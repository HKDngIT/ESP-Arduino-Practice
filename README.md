# ESP-Arduino-Practice

This repository contains my practice projects, experiments, and test code using ESP32 and Arduino platforms.

## Purpose

The goal of this repository is to:

* Learn and explore ESP32 and Arduino development
* Test hardware components and sensors
* Experiment with different libraries and communication protocols
* Save small example projects and proof-of-concept implementations
* Document my learning progress

## Notes

Most projects in this repository are intended for learning and testing purposes. Code quality, structure, and documentation may vary between examples.

## Hardware

Examples may use different hardware components, including:

* ESP32 development boards
* Arduino boards
* LCD/OLED displays
* Sensors and modules
* LEDs, relays, and other peripherals
